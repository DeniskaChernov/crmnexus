# 🖨️ Исправление: Артикул обрезается при печати этикетки

## ❌ Проблема:

При печати этикетки артикул обрезался:
- Показывалось: "4342*" (обрезано)
- Должно быть: "4342" (полностью)

На скриншоте видно что артикул не помещается и обрезается.

---

## 🔍 Причина:

В стилях для печати был слишком крупный шрифт и `overflow: hidden` обрезал текст:

```css
/* БЫЛО: */
.art { 
    font-size: 13pt;          /* ← Слишком крупно для некоторых артикулов */
    overflow: hidden;         /* ← Обрезает текст */
    text-overflow: ellipsis;  /* ← Добавляет ... */
}
```

**Результат:**
- Если артикул длинный → обрезается
- Добавляется многоточие (...)

---

## ✅ Решение:

### ВАЖНО: Стикер фиксированного размера 40×30mm

- ❌ **НЕ переносим текст на новую строку** (стикер увеличится и не поместится в область печати)
- ✅ **Уменьшаем шрифт артикула**
- ✅ **Делаем overflow: visible** чтобы текст был виден полностью

### Изменения в стилях:

**Файл:** `/components/Warehouse.tsx`, функция `printLabel` (строка ~1346)

```css
/* СТАЛО: */
.art { 
    font-size: 10pt;          /* ← Уменьшено с 13pt до 10pt */
    line-height: 1;           /* ← Плотно */
    white-space: nowrap;      /* ← Одна строка (не переносим!) */
    max-width: 38mm;          
    overflow: visible;        /* ← Виден полностью, не обрезается */
    text-overflow: clip;      /* ← Без многоточия */
}

.client {
    font-size: 8pt;           /* ← Уменьшено с 10pt до 8pt */
    white-space: nowrap;      /* ← Одна строка */
    overflow: visible;        
    text-overflow: clip;      
}

.date {
    font-size: 8pt;           /* ← Уменьшено с 9pt до 8pt */
}
```

## 🎯 Что изменилось:

### 1. **Уменьшен размер шрифта артикула**
- 13pt → **10pt** (на 23% меньше)
- ✅ Теперь помещается на одной строке
- ✅ НЕ переносится вниз (стикер остается 40×30mm)

**Результат:** Артикулы видны полностью без увеличения размера стикера

---

### 2. **Изменен overflow**
- ❌ Было: `overflow: hidden` (обрезало)
- ❌ Было: `text-overflow: ellipsis` (добавляло ...)
- ✅ Стало: `overflow: visible` (показывает полностью)
- ✅ Стало: `text-overflow: clip` (без многоточия)

**Результат:** Текст виден полностью, не обрезается

---

### 3. **Оптимизированы размеры всех элементов**
- Артикул: 13pt → **10pt** ⬇️
- Клиент: 10pt → **8pt** ⬇️
- Дата: 9pt → **8pt** ⬇️
- Вес: **26pt** (без изменений - самое важное!) 🔥

**Результат:** Все помещается на стикере 40×30mm

---

### 4. **Сохранен white-space: nowrap**
- ✅ `white-space: nowrap` - текст НЕ переносится
- ✅ Стикер не увеличивается в высоту
- ✅ Физический размер остается 40×30mm
