## 🎯 Workflow: Как использовать аналитику

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  1️⃣  ОТКРЫТЬ АНАЛИТИКУ                                     │
│      └─ Главное меню → Аналитика (📈 иконка)              │
│                                                             │
│  2️⃣  ПРИМЕНИТЬ ФИЛЬТРЫ (опционально)                       │
│      └─ Кнопка "Фильтры" → выбрать период                 │
│      └─ Пресеты: Неделя / Месяц / 90 дней                 │
│                                                             │
│  3️⃣  ИЗУЧИТЬ ОБЗОР                                         │
│      └─ Velocity, Retention, Cycle, Customers              │
│      └─ AI-инсайты для быстрого понимания                 │
│                                                             │
│  4️⃣  ДЕТАЛЬНЫЙ АНАЛИЗ ПО ВКЛАДКАМ                          │
│      ├─ ABC → топ-20% клиентов                            │
│      ├─ RFM → сегментация для персонализации              │
│      ├─ Когорты → LTV для планирования CAC                │
│      ├─ Скорость → узкие места в воронке                  │
│      ├─ Активность → оптимальное время контакта           │
│      └─ Прогноз AI → планирование на квартал              │
│                                                             │
│  5️⃣  ЭКСПОРТ ДЛЯ ОТЧЕТОВ                                    │
│      └─ Кнопка "Экспорт" → CSV для Excel                  │
│                                                             │
│  6️⃣  ДЕЙСТВИЯ НА ОСНОВЕ ДАННЫХ                             │
│      ├─ Реактивация At Risk клиентов                      │
│      ├─ Фокус на категорию A                              │
│      ├─ Оптимизация времени звонков                       │
│      └─ Корректировка целей по прогнозу                   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 🎓 Термины в одной картинке

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  ABC → Кто приносит 80% денег                              │
│        🟢 A (топ-20%) 🟡 B (средние) 🔴 C (остальные)     │
│                                                             │
│  RFM → Оценка клиента                                      │
│        R (давно?)  F (часто?)  M (много?)                 │
│        Scores: 1-4, где 4 = лучше                          │
│                                                             │
│  Когорта → Группа по месяцу 1-й покупки                    │
│            Январь 2024 = все кто купил впервые в янв       │
│                                                             │
│  LTV → Lifetime Value                                      │
│        Сколько $ принесет клиент за всё время              │
│                                                             │
│  Retention → Удержание                                     │
│              % клиентов вернувшихся снова                  │
│                                                             │
│  Velocity → Скорость генерации выручки                     │
│             (Кол-во × Чек × WinRate) / Цикл = сум/день    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
