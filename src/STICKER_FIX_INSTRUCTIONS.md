### 1. **Серверная часть** (`/supabase/functions/server/index.tsx`)
✅ Добавлено сохранение поля `stickerClient` в endpoint `POST /shipments`:
```typescript
const { date, note, items, status, warehouse, dealId, stickerClient } = body;
// ...
stickerClient: stickerClient || '',
```

### 2. **Создание отгрузки из сделки** (`/utils/crm/shipmentHelpers.ts`)
✅ Добавлена передача названия компании при создании отгрузки:
```typescript
stickerClient: clientName, // Company name for sticker printing
```

✅ Добавлена дата для каждого товара:
```typescript
date: new Date().toISOString(),
```

### 3. **Улучшен стикер** (`/components/Warehouse.tsx`)
✅ Улучшена визуализация:
- Название компании: **жирнее** (font-weight: 600)
- Размер шрифта компании: **увеличен** (10pt)
- Оптимизированы отступы для лучшего размещения

✅ Добавлено логирование для отладки:
```typescript
console.log('Printing label:', { item, clientName });
```

## Как тестировать

### Вариант 1: Создание отгрузки из сделки

1. Откройте **Сделки** → выберите сделку
2. Нажмите **"Создать отгрузку"**
3. Система автоматически подтянет название компании
4. Перейдите в **Склад** → вкладка **"Отгрузки"**
5. Найдите созданную отгрузку (статус: Черновик)
6. Нажмите кнопку **печати** 🖨️ на любом товаре
7. ✅ **Название компании должно быть вверху стикера**

### Вариант 2: Создание отгрузки вручную

1. Откройте **Склад** → вкладка **"Отгрузки"**
2. Нажмите **"+ Новая отгрузка"**
3. Заполните поля:
   - **Клиент / Примечание**: "Тестовая компания"
   - **Фирма на стикер**: **"ООО Тестовая"** ← ВАЖНО!
4. Добавьте товары
5. Нажмите **"Сохранить черновик"**
6. Нажмите кнопку печати 🖨️ на товаре
7. ✅ **"ООО Тестовая" должна быть вверху стикера**

### Проверка логов (если не работает)

1. Откройте **DevTools** (F12)
2. Перейдите на вкладку **Console**
3. Нажмите печать стикера
4. Найдите лог: `Printing label: { item: {...}, clientName: "..." }`
5. Проверьте что `clientName` **не пустой и не undefined**

## Если проблема осталась

### Проверьте:

1. **Очистите кеш браузера** (Ctrl+Shift+R)
2. **Проверьте что старые отгрузки обновлены:**
   - Откройте черновик отгрузки
   - Заполните поле "Фирма на стикер"
   - Сохраните снова

3. **Для отгрузок из сделок:**
   - Проверьте что у сделки указана компания (Company)
   - Пересоздайте отгрузку заново

4. **Проверьте консоль на ошибки:**
   - Красные ошибки в Console
   - Ошибки в Network (запросы к API)
